| Ctrl+/      | 添加/删除行注释 |
| ----------- | --------------- |
| Shift+Alt+A | 添加/删除块注释 |
| Shift+Alt + ↓ / ↑ | 向上/向下复制当前行 |
| Ctrl+X | 剪切当前行 |
| Alt+ ↑ / ↓        | 向上/向下移动当前行 |
| Home      | 转到行首 |
| End       | 转到行尾 |
| Ctrl+Home | 转到页首 |
| Ctrl+End  | 转到页尾 |
| Ctrl+↑ / ↓        | 向上/向下滚动 |
| Alt+PgUp / PgDown | 向上/向下翻页 |
